member/2

member(Queue::pqueue(), Item::queue_entry() | any() | {any(), any}) -> boolean()

Queue: Queue to search.
Item: Item to search for. Can be {Value, Priority} or Value.

returns: Return.

Checks if an item is part of the queue.

The item to search for may have a priority specified with it.

If the priority is the atom 'any' then it will search to see if the value is present without checking the priority. If a priority is specified, it will search for both a matching value and priority. Otherwise, it will only check for a matching value.
